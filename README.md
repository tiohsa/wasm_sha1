# wasm_sha1

SHA-1 Webassembly module

### Installing

```
npm install
cd template
npm install
```

## Running the dev server

```
wasm-pack build
npm start
```

## Release build

```
wasm-pack build --release
```

## Running the tests

```
wasm-pack test --headless --firefox
```
